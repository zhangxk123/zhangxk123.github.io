做自己

  • eddy
  • 0 Minutes
  • 2017年5月23日

人的一生在困惑的时候往往不知道该怎么办，经常会发问到底该怎么做。究其原因其实只是想寻求一个捷径，寻求一个关乎于人生或者只是某一件事的捷径。

举栗子：我们看心灵鸡汤以寻求一个获得完美心灵的捷径，查阅育儿资料文章以寻求一个成为完美父母的捷径，吃各种保健食品查阅养生文章以寻求一个获得好身体的捷径。
其实关于怎么做我最近突然想到了一个浅显无比却又经常被人们忽略掉的一个道理。人生没有捷径，唯一知道该怎么做的其实就只有你自己

人的一生太多杂念,开心就好，简单就好
