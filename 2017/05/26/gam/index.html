第七班

  • eddy
  • 0 Minutes
  • 2017年5月26日

从记事起，就被送到幼儿园了，
那时候也不知道朋友这个概念（现在才发现那时候成为朋友是多么的容易），
被大人以成长的说法送到一个有并排房间和操场的，有陌生大人的地方，（他们无暇照顾我们也是情理之中）

步入社会2年了，只有部分发小和高中基友还联系着，
人长大了，多年的经验使我们带上了有色眼睛，明白了人和人之间差距是如此之大，
弱肉强食的世界，一定要小心翼翼，不能掉以轻心，要学会保护自己，
慢慢的，朋友越来越少，或者这才是最开始的样子，
只是孩子气拯救了我们，让我们单纯的度过人生最快乐的时光，

越来越理智，也越来越。。。怎么说越冷血？

充满希望的、自信的少年，终究变成那个高不成低不就的普通大叔，
原来我也如此，太盲目自信了呀，
出生已决定了自己的位置，

贫穷让我有一些自卑，错过很多事情，
很后悔，但是也使我认清了自己，好死不如赖活着，
也只有在游戏里，这个相对公平的环境里，可以让我不顾一切

慢慢的，被套上标签，各种标签，有喜欢看到的，不喜欢的，这才是别人眼中的我，
政治正确的说，不要活在别人的眼光中，
正因为有别人的反馈，才让我成长，我应该感谢指正我的人，

未来不可避免的阶级固化，虽然有点悲观了些，
但是也许是一条出路，毕竟人人不平等，每个人都有自己的位置，自己的价值，好比游戏，这是一个团队，一个家庭，一个社会

这种发挥自己独特作用，贡献自己一份力量，是多么令人欣慰，令人向我 :）
